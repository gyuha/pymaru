# pymaru

## Install

```bash
sudo apt-get install python3-pyqt5
sudo apt-get install pyqt5-dev-tools
sudo apt-get install qttools5-dev-tools
```

## start

```bash
python3 -m venv venv
source venv/bin/active
pip install -r requirements.txt
```
